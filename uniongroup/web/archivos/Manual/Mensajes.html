Mensajes

8.1 Error

Este tipo de mensajes indican una validación incorrecta en el sistema.

Dentro de esta categoría mensajes existe:

8.1.1 La cantidad a mover no puede ser mayor que la cantidad disponible.

El sistema arroja este mensaje cuando se intenta agregar a la lista de movimiento de producto, un número mayor al de los productos disponibles.

8.1.2 Error de validación

Esta serie de mensajes aparecen cuando existe de uno a seis errores de validación al cargar un archivo excel en el módulo de facturación.

8.1.3 Errores de validación

Este mensaje aparece cuando existe más de seis errores de validación al cargar un archivo excel en el módulo de facturación. Para acceder a esta lista se debe seleccionar si en el mensaje Hay más de 6 errores de validación. ¿Desea ver todos los errores?

8.2 Advertencia

Este tipo de mensajes indican una validación incorrecta por parte de los datos ingresados por el usuario.

Dentro de esta categoría mensajes existe:

8.2.1 Nombre de usuario o clave no coinciden

Este mensaje advierte al usuario que ha ingresado incorrectamente el nombre de usuario y/o la clave.

8.2.2 Debe seleccionar los datos requeridos
8.3 Éxito

Este tipo de mensajes indican que se realizo una función de manera exitosa.

Dentro de esta categoría mensajes existe:

8.3.1 Se insertó el pedido.

Este mensaje indica la confirmación del levantamiento de un pedido y el número del mismo, aparte de las direcciones de correo electrónico a las que se enviará la confirmación.

8.3.2 Se cargó el archivo correctamente.

Módulo de facturación, carga de archivo .xlsx.

8.4 Información

Este tipo de mensajes informan al usuario datos importantes de las funcionalidades del sistema.

Dentro de esta categoría mensajes existe:

8.4.1 Procesando su solicitud, por favor espere…

Este mensaje indica que la solicitud está siendo procesada por el sistema. Al terminar el tiempo de espera desaparecerá automáticamente.

8.4.2 La consulta no devolvió ningún valor.

Este mensaje indica que, con los parámetros de consulta seleccionados, no existe ningún valor. Se deberá cerrar el mensaje haciendo click en el botón Aceptar y seleccionar otros parámetros de consulta.

8.4.3 Debe primero realizar una búsqueda para exportar el Excel.

8.4.4 Debe seleccionar los datos requeridos.

Este mensaje indica que aún hay datos incompletos en el menú de parámetros de consulta. Se deberán seleccionar todos los datos requeridos para realizar una consulta.

8.5 Decisión

Este tipo de mensajes preguntan al usuario si desea realizar la acción seleccionada.

Dentro de esta categoría mensajes existe:

8.5.1 ¿Desea cerrar la sesión?

Este mensaje pregunta al usuario si desea cerrar la sesión. Para cerrar la sesión se deberá hacer click en el botón Sí.

8.5.2 ¿Desea exportar la página actual o todos los registros existentes?

Este mensaje hace la pregunta al usuario si quiere exportar los 100 datos mostrados en la página actual o el total de datos resultantes de la consulta.

8.5.3 ¿Estás seguro que deseas mover los productos seleccionados?

Este mensaje pregunta al usuario si desea mover los productos previamente seleccionados en la funcionalidad de Movimientos (realizar un pedido).

8.5.4 Hay más de 6 errores de validación. ¿Desea ver todos los errores?

Cuando se realiza la carga de un archivo .xlsx y existen más de 6 errores de validación en el formato del archivo a cargar, se muestra este mensaje que pregunta al usuario si desea ver todos los errores de validación.
